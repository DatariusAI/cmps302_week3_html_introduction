# CMPS 302 - Week 3: HTML Introduction
## Alps Tourism Website

A comprehensive tourism website about the Alps mountain range, showcasing geography, ecology, tourism destinations, and multimedia content. This project demonstrates semantic HTML5 structure, CSS styling, and responsive web design principles.

## Getting Started

### View Online
Visit the live site: https://DatariusAl.github.io/cmps302_week3_html_introduction/

### Run Locally
1. Clone the repository:
```bash
   git clone https://github.com/DatariusAl/cmps302_week3_html_introduction.git
```

2. Navigate to the project directory:
```bash
   cd cmps302_week3_html_introduction
```

3. Open index.html in your browser or simply double-click index.html in your file explorer.

### File Structure Notes
- CSS files must be in the css folder (lowercase)
- Gallery images in Gallery folder
- Content images in Images folder
- All HTML files in root directory
