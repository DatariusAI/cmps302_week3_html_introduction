# CMPS 302 - Week 3: HTML Introduction
## Alps Tourism Website

A comprehensive tourism website about the Alps mountain range, showcasing geography, ecology, tourism destinations, and multimedia content. This project demonstrates semantic HTML5 structure, CSS styling, and responsive web design principles.

## Content Overview

### Eight Alpine Countries

1. Austria - Republic of Austria, Capital: Vienna
2. France - French Republic, Capital: Paris
3. Germany - Federal Republic of Germany, Capital: Berlin
4. Italy - Italian Republic, Capital: Rome
5. Liechtenstein - Principality of Liechtenstein, Capital: Vaduz
6. Monaco - Principality of Monaco, Capital: Monaco
7. Slovenia - Republic of Slovenia, Capital: Ljubljana
8. Switzerland - Swiss Confederation, Capital: Bern

### Key Facts

- Highest Peak: Mont Blanc - 4,810m (15,781 ft)
- Range Length: 800 km (500 mi)
- Width: 200 km (120 mi)
- Plant Species: 13,000+
- Wildlife Species: 30,000+
- Peaks Above 4,000m: 100+
